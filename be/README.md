# Aidnara AI - Backend

This is the backend service for the Aidnara AI project, providing APIs for campaign management, donation tracking, AI impact evaluation, and smart contract synchronization.

## Tech Stack
* Language: Go
* Framework: Fiber (v2)
* Database: PostgreSQL
* Database ORM: sqlc (Type-safe SQL)
* Blockchain: go-ethereum (Geth)

## Prerequisites
* Go (1.20 or newer)
* PostgreSQL database

## Getting Started

1. Environment Setup:
   Copy the example environment file and configure the necessary variables.
   ```bash
   cp .env.example .env
   ```
   Ensure you provide a valid `DATABASE_URL` pointing to your local PostgreSQL instance and set your `GEMINI_API_KEY`.

2. Database Setup:
   Apply the SQL schemas found in `db/schema.sql` to your PostgreSQL database.
   ```bash
   psql -U postgres -d aidnara -f db/schema.sql
   ```

3. Download Dependencies:
   ```bash
   go mod tidy
   ```

4. Run the Server:
   ```bash
   go run main.go
   ```
   The server will start on port 3000 by default (or the port specified in your .env file).

## Project Structure
* `handlers/` - Contains the HTTP route handlers for the API endpoints.
* `services/` - Contains integrations with external services (Google Gemini, Web3/Onchain validation, Local Storage).
* `db/` - Contains raw SQL queries (`query.sql`, `schema.sql`) and code generated by sqlc.
* `uploads/` - Local directory created dynamically to store uploaded files (images, proofs).
